## 技术栈共性

- **身份认证**：DID + Agent Card + 身份联邦
- **策略执行**：IBAC（意图驱动）+ 自然语言策略
- **可观测性**：otel 标准 + 全链路追踪
- **审计取证**：TEE + 区块链存证
- **部署方式**：私有化 / 混合云 / 边缘节点
- **框架兼容**：Dify / LangChain / MCP 协议
