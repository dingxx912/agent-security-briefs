# 智能体安全产品速查包

> 一个给 AI agent 用的产品速查档案。下载即用，不需要安装任何东西。

## 这是什么

把公司智能体安全产品线的核心信息整理成 AI 可读的 Markdown 文档。售前、架构师、市场部不需要翻 PRD，下载解压后直接对着文件夹问 AI 就行。

## 覆盖产品

- 智能体身份安全
- AI-BOM
- 智能体安全、智能体安全网关、ANS（陆续补充中）
- 海外版

## 怎么用

1. 下载 ZIP 并解压
2. 在文件夹里打开 Claude Code / Codex / 任何支持 CLAUDE.md 的 AI 工具
3. 直接问，比如：
   - "AI-BOM 是干什么的？跟竞品比有什么优势？"
   - "智能体身份安全的架构分几层？"
   - "最近数据泄露热点，哪个产品能蹭？"

AI 会自动读 `CLAUDE.md`，根据你的角色（售前/架构师/市场）用不同方式回答。

## 怎么维护

- 产品状态变了 → 改对应文件夹的 `tasks.md`
- 客户问了新问题 → 更新 `sales-brief.md`
- 有新热点事件 → 更新 `market-brief.md`
- 答不上来的问题自动记入 `qa-log.md`

## 文件结构

```
├── CLAUDE.md                  ← AI 操作手册（Agent 首先读这个）
├── qa-log.md                  ← 未命中问题日志
├── 0-产品全景/                 ← 产品矩阵、关系图、术语表
├── 1-智能体身份安全/            ← 产品文档 + 原始 PRD
├── 3-AI-BOM/                  ← 产品文档 + 原始 PPT
└── shared/                    ← 竞品对比、技术总图、热点匹配
```
