## 场景配合关系

| 客户场景 | 涉及产品 |
|---|---|
| "搞不清公司有多少智能体和模型在用" | AI-BOM（先盘点） |
| "智能体违规调用核心系统" | 身份安全（管权限）+ 安全网关（卡入口） |
| "有人绕过网关直接调模型"（影子 AI） | AI-BOM（发现）+ 安全网关（管控） |
| "需要给监管和审计出报告" | 身份安全（全链审计）+ AI-BOM（资产台账） |
| "海外业务合规" | AI-BOM 海外版 + 安全网关（共享） |
