# 术语表

## 核心概念

| 术语 | 全称 | 一句话解释 |
|---|---|---|
| **AI-BOM** | AI Bill of Materials | AI 物料清单，一份记录企业所有 AI 资产（模型、智能体、工具、数据）的结构化台账 |
| **SPDX 3.0** | Software Package Data Exchange 3.0 | 国际通用的软件物料清单标准，AI-BOM 基于此标准扩展 AI 资产描述 |
| **IBAC** | Intent-Based Access Control | 意图驱动访问控制，不是看"你是谁"而是看"你想干什么"来决定给不给权限 |
| **HITL** | Human-In-The-Loop | 人机协作共治，关键决策由人来确认，AI 不能全自动做高风险操作 |
| **Agent Card** | 代理卡 | 给每个智能体发的数字身份卡，记录它能做什么、属于谁、权限范围 |
| **MCP** | Model Context Protocol | 模型上下文协议，智能体调用外部工具的通用接口标准 |
| **影子 AI** | Shadow AI | 员工或部门绕过公司 IT 管理，自己用 AI 工具或搭智能体（类似当年的"影子 IT"） |
| **TEE** | Trusted Execution Environment | 可信执行环境，硬件级的安全隔离区，保证代码和数据不可篡改 |
| **DID** | Decentralized Identity | 去中心化身份，不依赖单一中心机构来验证身份 |
| **ANS** | Agent Nexus | 智能体编排总控层，统一管理调度所有智能体相关能力 |

## 六维信任框架（身份安全产品核心）

| 维度 | 一句话 |
|---|---|
| 身份可信 | 你是谁？（人和智能体都要验证） |
| 意图对齐 | 你想干嘛？（任务目标有没有被篡改） |
| 生成有界 | 你能看/说什么？（数据和内容的边界） |
| 行为可控 | 你能做什么？（工具调用和系统操作的权限） |
| 链路可审 | 你做了什么？（全链路记录，能重放） |
| 责权可溯 | 谁负责？（出了问题怎么追责，链上存证） |
