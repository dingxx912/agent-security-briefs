# 公司智能体安全产品线全景

## 一句话定位

为企业的 AI 智能体（Agent）提供从开发、部署到运营的全生命周期安全防护与资产治理。

## 产品矩阵

```
        智能体身份安全 ──┐
        智能体安全     ──┼── 智能体安全网关 ── ANS（Nexus，编排总控层）
        AI-BOM        ──┘

海外版：AI-BOM（主力）+ 部分身份安全 + 部分智能体安全 + 安全网关（共享）
```

## 各产品一句话

| 产品 | 解决什么问题 |
|---|---|
| **智能体身份安全** | 给每一个 AI 智能体"上户口"，管住谁能调用什么、能做什么 |
| **智能体安全** | （待补充） |
| **AI-BOM** | 自动发现企业里所有 AI 资产（模型/智能体/工具/API），建台账防"影子 AI" |
| **智能体安全网关** | 所有智能体调用的统一入口，做实时鉴权、限流、监控 |
| **ANS（Nexus）** | 智能体的总控编排层，统一管理调度 |

## 产品间关系

- **AI-BOM** 负责"盘点清"——先搞清楚企业有什么 AI 资产
- **智能体身份安全** 负责"管住谁"——给每个资产和人建立身份和权限
- **智能体安全网关** 负责"卡住口"——所有调用必须过网关
- **智能体安全** 负责"看住行为"——智能体行为分析和风险检测
- **ANS** 负责"统起来"——把以上能力编排成统一的管控平面

## 海外版差异

海外版以 AI-BOM 为主力产品，搭配部分身份安全和智能体安全模块，共享同一套安全网关底座。差异主要在合规标准（国内 vs 国际）和部署方式。
