# 海外版产品组合

## 组合定位

以 AI-BOM 为主力产品，搭配部分智能体身份安全和智能体安全模块，共享同一套智能体安全网关底座。

## 与国内版的差异

| 维度 | 国内版 | 海外版 |
|---|---|---|
| 主力产品 | 全产品矩阵并行 | AI-BOM 为主 |
| 合规标准 | 等保/关基/国标 | SPDX 3.0 / 国际标准为主 |
| 部署方式 | 私有化为主 | 混合云/本地部署 |

## 产品优先级

| 优先级 | 产品 | 说明 |
|---|---|---|
| P0 | AI-BOM | 主力，适应国际合规标准的独立版本 |
| P1 | 智能体安全网关 | 共享底座 |
| P2 | 智能体身份安全（简化版） | 裁剪为海外适用模块 |
| P2 | 智能体安全（简化版） | 裁剪为海外适用模块 |
